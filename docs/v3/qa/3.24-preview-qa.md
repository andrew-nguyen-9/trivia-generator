## Findings to fix (flagged — outside this segment's owned files)

These live in game components owned by other v3 segments. Per the conflict
firewall this segment records but does not edit them.

1. **`/board` — broken ARIA grid (critical).** `frontend/components/BoardGame.tsx`
   renders `<div role="grid">` with 25 direct `role="gridcell"` children and **no
   `role="row"` layer**. WAI-ARIA requires `grid → row → gridcell`; the missing row
   trips `aria-required-children` (grid) and `aria-required-parent` (×25 cells).
   Regression from §3.1 (PR #13) keyboard-nav grid. Fix: wrap each 5-cell row in
   `role="row"`. → **§3.1 Board owner.**
2. **`/clock` — one serious `color-contrast`** node. → §3.2 Clock owner.
3. **Lighthouse a11y −2 on `/`:** `heading-order` (non-sequential headings) +
   `label-content-name-mismatch` (visible label ≠ accessible name). → home/shell.

None block the preview; all are pre-existing component a11y debt now made visible.
